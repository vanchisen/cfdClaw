- THERMO=0 in Nektar2.5D build flags means: do NOT build heat-transfer (energy) equation capability.
- SPM=0 means: do NOT build Smoothed Profile Method capability (used for flow with moving particles).
- Ur sweep runs: if any case blows up, decrease DT in the corresponding *.rea and relaunch.
- PhaseField/Mesh mesh conversion pipeline: gmsh2grd converts Gmsh .msh -> .grd; gg_curve converts .grd -> Nektar *.rea mesh; gg_boundary handles boundary conditions.
